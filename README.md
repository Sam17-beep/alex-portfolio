# Portfolio

## How to Deploy

### Prerequisites

- Node.js installed
- pnpm installed

### Steps

1. Install dependencies:

   ```sh
   pnpm install
   ```

2. Build the project:

   ```sh
   pnpm build
   ```

3. Deploy to Vercel:

   ```sh
   vercel --prod
   ```

### Development

Run the dev server locally:

```sh
pnpm dev
```
